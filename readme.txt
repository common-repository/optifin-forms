== Description ==

OPTIFIN Forms Plugin provides loan and other financial calculators for our affiliate partners.

These calculators and forms are included:

Kreditrechner
Umschuldungsrechner
Haushaltsbudgetrechner
Hauskostenrechner
Nebenkostenrechner
Effektivzinssatzrechner
Kaufen oder Mieten Rechner
Anfrage Formular


Installation

To include the different calculators on your page, use the following shortcodes:
‘
[kreditrechner opti_pid=""]
[umschuldungsrechner opti_pid="" ]
[haushaltsbudgetrechner opti_pid="" ]
[hauskostenrechner opti_pid="" ]
[nebenkostenrechner opti_pid="" ]
[effektivzinssatzrechner  opti_pid="" ]
[kaufenodermietenrechner opti_pid="" ]
’

Fill in your personal OPTIFIN Partner ID like this: opti_pid="1234".

The calculators are customizeable in color:

color1 = "#ffffff"

color2 = "#ffffff"

color3 = "#ffffff"

color4 = "#ffffff"

There are two options to send the user to OPTIFIN:

1.) (Default) By click on the CTA Buttons, the User will be redirected to OPTIFIN where the lead is generated.

2.) By adding: include_inquiry_form="true" the lead form will appear directly under the calculator, when the user clicks on the CTA Buttons.
